# 客户管理页面 - 快速启动指南

## 访问页面

1. 启动后端服务（Spring Boot）：
   ```bash
   cd backend
   mvnw.cmd spring-boot:run
   ```

2. 启动前端服务（Vue 3）：
   ```bash
   cd frontend
   npm run dev
   ```

3. 在浏览器中访问：
   ```
   http://localhost:5173
   ```

4. 登录系统后，点击侧边栏的"客户管理"菜单项

## 功能使用

### 查看客户列表
- 页面加载后自动显示客户列表
- 默认每页显示 10 条记录

### 搜索客户
1. **按姓名搜索**：
   - 在"姓名"输入框中输入关键字
   - 点击"查询"按钮
   - 支持模糊匹配（例如：输入"张"可以找到"张三"、"张四"等）

2. **按手机号搜索**：
   - 在"手机号"输入框中输入完整手机号
   - 点击"查询"按钮
   - 必须输入完整的 11 位手机号

3. **重置搜索**：
   - 点击"重置"按钮清空所有搜索条件

### 新增客户
1. 点击右上角的"新增客户"按钮
2. 在弹出的对话框中填写客户信息：
   - 姓名（必填）
   - 手机号（必填，格式：1[3-9]开头的11位数字）
   - 身份证号（必填，格式：18位标准格式）
   - 性别（可选，选择男或女）
   - 地址（可选）
3. 点击"确定"按钮提交
4. 提交成功后，列表会自动刷新

### 编辑客户
1. 在客户列表中找到要编辑的客户
2. 点击该行的"编辑"按钮
3. 在弹出的对话框中修改客户信息
4. 点击"确定"按钮保存修改
5. 保存成功后，列表会自动刷新

### 删除客户
1. 在客户列表中找到要删除的客户
2. 点击该行的"删除"按钮
3. 在确认对话框中点击"确定"
4. 删除成功后，列表会自动刷新

**注意**：如果客户有关联的订单，删除操作会失败，系统会提示错误信息。

### 分页操作
1. **切换页码**：
   - 点击页码数字直接跳转
   - 点击"上一页"/"下一页"按钮

2. **修改每页条数**：
   - 点击"10 条/页"下拉框
   - 选择 10、20、50 或 100

3. **跳转到指定页**：
   - 在"前往"输入框中输入页码
   - 按回车键跳转

## 常见问题

### Q1: 为什么搜索不到客户？
A: 请检查：
- 姓名搜索是否输入了正确的关键字（支持部分匹配）
- 手机号搜索是否输入了完整的 11 位手机号（必须完全匹配）
- 是否点击了"查询"按钮

### Q2: 为什么无法新增客户？
A: 请检查：
- 所有必填字段是否都已填写
- 手机号格式是否正确（1[3-9]开头的11位数字）
- 身份证号格式是否正确（18位标准格式）
- 手机号或身份证号是否已被其他客户使用

### Q3: 为什么无法删除客户？
A: 可能的原因：
- 该客户有关联的销售订单，系统不允许删除
- 解决方案：先删除或取消该客户的所有订单，然后再删除客户

### Q4: 如何查看客户的订单历史？
A: 当前版本暂不支持直接查看客户订单历史，可以：
1. 前往"订单管理"页面
2. 使用客户姓名或手机号搜索相关订单
